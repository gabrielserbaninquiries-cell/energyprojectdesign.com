## A. AI as Co-Engineer (autonomous design)

### A.1 AI Project Designer (full autonomy)
- [ ] User încarcă fotografie teren + acte teren (carte funciară) → AI generează SF + DTAC + PT complet
- [ ] Iterare interactivă: "fă-o cu garaj suprateran", "adaugă piscină"
- [ ] Output: 50+ pagini documentație tehnică în 5 minute

### A.2 AI Compliance Officer
- [ ] Scanează oricât e proiect → detectează încălcări norme (Codul fiscal, Legea 10/1995, P118-99, etc.)
- [ ] Sugerează patch-uri textuale automate
- [ ] Generează raport audit-ready (pentru ISC, ANRE, etc.)

### A.3 AI Cost Estimator
- [ ] Input: m² + tip lucrare → estimare cost materiale + manoperă + termen execuție
- [ ] Comparare furnizori în timp real (Dedeman, Brico, Hornbach, Praktiker)
- [ ] Profit margin recommendări (per industrie)

## B. Blockchain & Web3

### B.1 Documents on chain (notarization)
- [ ] Hash SHA-256 al fiecărui document → commit pe Ethereum/Polygon
- [ ] Public verification (anyone can verify ownership)
- [ ] Time-stamping legal (eIDAS compliant)

### B.2 NFT certificates pentru autorizații
- [ ] Autorizație ANRE → NFT cu metadata semnat
- [ ] Transfer NFT → transfer autorizație (M&A friendly)

### B.3 Smart contracts pentru contracte clienți
- [ ] Contract în Solidity → plată automată milestone-uri
- [ ] Escrow nativ (Polygon) pentru acont
